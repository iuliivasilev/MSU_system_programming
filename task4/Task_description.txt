В качестве большого семестрового задания Task 4  предлагаются четыре направления:
 
1) http://cmcmsu.info/download/model.sql.interpreter.2005.pdf   Модельный SQL-интерпретатор
2) http://cmcmsu.info/download/model.lang.practical.task.pdf   Интерпретатор модельного языка программирования
3) http://cmcmsu.info/download/avst.gameserv.pdf   Игровой клиент (сервер выдается готовый)
4) http://cmcmsu.info/download/cpp.http.server.pdf  Модельный веб-сервер

Тем, кто выберет SQL-интерпретатор, можно использовать готовую библиотеку для работы с таблицами (одна таблица хранится в одном файле).
Эту библиотеку можно найти на сайте в разделе для 2 курса: http://cmcmsu.info/download/table.c.lib.zip
Библиотека написана на Си, можно самим написать классы-"обертки", чтобы использовать ее в ООП-стиле.

Игровой сервер.
Тем, кто выберет этот вариант задания, можно воспользоваться приложенной к письму 
версией сервера (написанной Н. В. Арефьевым) или сервером А.В. Столярова
(искать на сайтах, например  http://al.cs.msu.su/static/avst/manager-0.4.03.tgz).
Язык программирования играющих ботов можно дополнить новыми конструкциями.
При показе нужно демонстрировать игру между разными роботами (действующими
по различным стратегиям), написанными на вашем языке программирования.

Для  направления "интерпретатор ЯП"  вот возможные варианты языков:  http://cmcmsu.info/download/model.lang.practical.task.pdf
Вот еще пример вариантов от Л.Н. Кузиной:    http://al.cmc.msu.ru/files/kuzina_prac_task2_2009.pdf
Можно придумать свой вариант языка. Тогда его нужно описать в таком же стиле (синтаксис, семантика), как в предложенном документе выше.
   
Здесь пример реализации модельного ЯП, рассматриваемого на лекциях:   http://cmcmsu.info/2course/model.lang.parser.sample.htm
Веб-сервер : см. метод пособие + можно усложнить модельный  JavaScript-язык
